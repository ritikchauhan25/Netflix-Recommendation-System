## 📌 Project Overview

This project builds a **Netflix Movie Recommendation System** using **Singular Value Decomposition (SVD)** and **Collaborative Filtering** techniques. It recommends movies to users based on historical user ratings and preferences.

---

## 🎯 Objective

Develop a recommendation system capable of suggesting personalized Netflix movies by learning user preferences using collaborative filtering and matrix factorization.

---

## 📂 Dataset

Dataset Used:

- netflix_titles.csv

Dataset contains:

- Movie Title
- Genre
- Director
- Cast
- Country
- Release Year
- Rating
- Duration
- Description
